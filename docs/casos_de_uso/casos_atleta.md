# Historias de Usuario: Atleta (Versión MVP)

Este documento define todas las funcionalidades que el usuario de tipo "Atleta" podrá realizar dentro de la aplicación "Aria Training" para la primera versión (MVP).

---

### Gestión de Cuenta y Perfil

* **ID:** A-01
    * **Historia:** Como **nuevo usuario**, quiero **registrarme en la aplicación con mi correo y contraseña** para **crear mi perfil de atleta**.
* **ID:** A-02
    * **Historia:** Como **Atleta**, quiero **iniciar sesión con mis credenciales** para **acceder a mis rutinas y a mi perfil**.
* **ID:** A-03
    * **Historia:** Como **Atleta**, quiero **ver mi información básica de perfil** (nombre, correo) para **asegurarme de que mis datos son correctos**.
* **ID:** A-04
    * **Historia:** Como **Atleta**, quiero **poder editar mi nombre y cambiar mi contraseña** para **mantener mi cuenta actualizada y segura**.
* **ID:** A-05
    * **Historia:** Como **Atleta**, quiero **poder cerrar sesión** para **proteger la privacidad de mis datos**.

---

### Visualización y Ejecución del Entrenamiento

* **ID:** A-06
    * **Historia:** Como **Atleta**, quiero **acceder a un Dashboard optimizado para móvil** que me muestre inmediatamente el entrenamiento de hoy y un botón grande para comenzar.
* **ID:** A-07
    * **Historia:** Como **Atleta**, quiero **ver la lista de ejercicios agrupada por bloques** (calentamiento, biseries) para **entender la estructura de mi sesión**.
* **ID:** A-08
    * **Historia:** Como **Atleta**, quiero **registrar peso y repeticiones reales en cada serie** y marcarlas como completadas con un simple toque (tap).
* **ID:** A-09
    * **Historia:** Como **Atleta**, quiero **registrar mi RPE (Esfuerzo Percibido) y RIR (Reps en Reserva)** si mi entrenador lo solicita, para **ajustar la intensidad**.
* **ID:** A-10
    * **Historia:** Como **Atleta**, quiero **ver mis notas personales y las indicaciones del entrenador** en cada ejercicio para **no olvidar detalles técnicos**.

---

### Seguimiento de Progreso

* **ID:** A-11
    * **Historia:** Como **Atleta**, quiero **ver un historial visual de mis levantamientos** en cada ejercicio para **saber cuánto peso usar hoy**.
* **ID:** A-12
    * **Historia:** Como **Atleta**, quiero **recibir feedback visual (check verde)** al completar una serie para **sentir progreso durante la sesión**.
